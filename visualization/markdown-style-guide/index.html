    Markdown 编写规范

    统一 Markdown 编写风格有助于文档一致性和可读性。

    1. 标题层级

    使用 # 至 ###### 表示 H1 到 H6，通常从 H2 (##) 起作为正文标题。

    2. 代码块

    使用三个反引号包裹代码，并标注语言：

    ```python
print("Hello World")
````
