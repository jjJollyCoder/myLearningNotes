
# Markdown 编写规范

统一 Markdown 编写风格有助于文档一致性和可读性。

## 1. 标题层级

使用 `#` 至 `######` 表示 H1 到 H6，通常从 H2 (`##`) 起作为正文标题。

## 2. 代码块

使用三个反引号包裹代码，并标注语言：

<pre>
```python
print("Hello World")
````

</pre>

## 3. 列表

* 使用无序列表表示要点
* 使用有序列表表示步骤

## 4. 引用块

> 用于引用、提示或重要说明。

## 5. 表格

| 模块         | 描述     |
| ---------- | ------ |
| Flask      | Web 框架 |
| SQLAlchemy | ORM 工具 |

## 6. 图片和链接

* 图片: `![描述](图片链接)`
* 链接: `[文本](链接地址)`
